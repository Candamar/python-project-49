## Synopsis

Brain Games are five games that you can play on the console. Made by Alexey 
Khvostantsev as the first project in Hexlet course of Python.

## Installation

`make package-install`

## Run games

1. `brain-even`
2. `brain-calc`
3. `brain-gcd`
4. `brain-progression`
5. `brain-prime'
